# Programacao_web_2026I

--Descreva as motivações para o desenvolvimento de aplicações baseadas em internet ?--
Uma das motivações é acessibilidade universal da aplicação. As pessoas estão todas conectadas, uma aplicação baseada em web fornece
uma forma de incluí-las nos sistemas e afins.

--Quais as funções de um servidor de aplicação?--
Ele intermedia o usuário ao banco de dados. Ele traz o ambiente para que as aplicações funcionem com seu devido fluxo de trabalho.

--Quais as diferenças das camadas de frontend e backend ?--
O front-end é o que vemos e interagimos, com a interface e todas suas funcionalidades. O back-end é o que sustenta, o que faz as 
funcionalidades, de fato, funcionarem.
